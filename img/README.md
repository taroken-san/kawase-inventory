# カタログ商品画像フォルダ

在庫アプリ（inventory.html）のカタログに表示する商品写真を置く場所です。

## 使い方
1. 画像をこの `img/` フォルダに入れる（推奨：横幅800px前後・jpg/png・1MB以下）
2. ファイル名は半角英数で品番ベースにする（例：`fs-1g.jpg` `5-s.jpg`）
3. アプリの商品「編集」→「商品画像」欄に `img/ファイル名.jpg` と入力
   - メーカー公式画像のURL（https://...）を直接入れてもOK
4. このフォルダごとGitHubへpushすると公開URLに反映される

公開URL例: https://taroken-san.github.io/kawase-inventory/img/fs-1g.jpg
